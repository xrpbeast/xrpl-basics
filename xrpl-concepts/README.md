# XRPL Concepts

Core concepts and examples for XRP Ledger development.

## Lessons

0. [Prerequisites](./00-prerequisites.md) - Install Node.js, npm, VS Code
1. [What is the XRP Ledger?](./01-what-is-xrpl.md) - Core properties and strengths
2. [Hello World](./02-hello-world.md) - Connect and fetch ledger info
3. [History](./03-history-of-xrpl.md) - Origins, timeline, and evolution
4. [Features Overview](./04-features-overview.md) - Payments, DEX, NFTs, escrow, and more

## Topics Covered

- **Accounts** - Address generation, key pairs, account settings
- **Transactions** - Payment types, signing, submission
- **Trust Lines** - Token issuance, holding issued currencies
- **NFTs** - Minting, trading, burning NFTokens
- **DEX** - Offers, order books, auto-bridging
- **Escrow** - Time-based and condition-based escrows
- **Payment Channels** - Off-ledger micropayments
