# What is the XRP Ledger?

A decentralized, permissionless, open source blockchain for building and transacting.

## Core Properties

| Property | Description |
|----------|-------------|
| **Decentralized** | No single entity controls it. 100+ validators confirm transactions globally |
| **Permissionless** | Anyone can participate without approval |
| **Open Source** | Full codebase transparency, anyone can review/contribute |

## Key Strengths

### Reliability
- 10+ years continuous uptime
- 80M+ ledgers closed
- 2.5B+ transactions processed

### Performance
- **Speed**: 3-5 second confirmation
- **Throughput**: Up to 3,400 TPS
- **Cost**: Fractions of a cent per transaction

### Sustainability
- Consensus mechanism is 61,000x more energy efficient than Bitcoin's proof-of-work

### Developer Experience
- Straightforward API
- Official SDK (xrpl.js)
- Well-documented primitives
